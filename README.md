Proyecto web que presenta la visión de una federación europea integrada para el horizonte **2040–2060**. Un sitio estático construido con tecnologías modernas, optimizado para SEO y listo para producción.

---

## 📋 Descripción

Este es un **proyecto de aprendizaje habitual** que explora una propuesta de federación europea realista. El contenido aborda:

- Sistema político federal con democracia directa
- Defensa unificada (Ejército Europeo + UMEE)
- Infraestructura conectada (EuroBananas)
- Educación compartida con *Lingua Franca* europea
- Símbolos federales que complementan identidades locales
- Vivienda garantizada en 22 años
- Sistema de pensiones portables y estables

> 🤖 El contenido ha sido generado y refinado con **ChatGPT** y **Grok** de forma iterativa.

## 🛠️ Stack Tecnológico

- **Astro** 5.16 — Framework SSG
- **React** 19 — Componentes interactivos
- **Tailwind CSS** 4 + **daisyUI** 5 — Estilos y temas
- **Tabler Icons** — Iconografía
- **GSAP** 3.14 — Animaciones (landing page)
- **Tipografía**: Cinzel (headings) + Inter (body)
- **Gestor de paquetes**: pnpm

## 📁 Estructura

```
src/
├── components/      # Header, Footer, Section…
├── layouts/         # Layouts de página
├── pages/           # Rutas Astro (index, politica, defensa…)
└── styles/          # Estilos globales
```

Páginas: `index` · `politica` · `defensa` · `infraestructura` · `educacion` · `simbolos` · `vivienda` · `intro`

## 🎨 Características

- **5 temas daisyUI**: light, dark, corporate, business, luxury
- **Responsive**: Mobile-first
- **Accesibilidad**: HTML semántico, ARIA labels, contraste WCAG
- **SEO**: Meta tags, Open Graph, sitemap XML
- **Animaciones**: GSAP en landing page

## 🚀 Comandos

| Comando | Acción |
|:--|:--|
| `pnpm install` | Instala dependencias |
| `pnpm dev` | Servidor de desarrollo en `localhost:4321` |
| `pnpm build` | Compila para producción en `./dist/` |
| `pnpm preview` | Previsualiza la build localmente |
